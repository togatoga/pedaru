# Contributing to Pedaru

Thank you for considering contributing to Pedaru! This project is still under active development, and contributions are welcome.

## Adding a new feature

If you want to add a new feature, please open an issue first to discuss your ideas. This helps ensure your contribution aligns with the project's direction and avoids duplicated effort.

## Bug fixes

For typos or small bug fixes, feel free to submit a pull request directly. For larger bug fixes, please open an issue first to discuss the approach.
